Symmetric Encryption and Decryption using mcrypt on Kali Linux

Overview
This project demonstrates symmetric encryption and decryption using mcrypt on Kali Linux.

Features
Symmetric encryption and decryption using mcrypt

Bash
Supports rijndael-256 encryption algorithm
Getting Started

Prerequisites
Kali Linux 2024 or higher
mcrypt extension for PHP
Bash shell

Installation
Clone the repository: git clone https://github.com/your-username/your-repo-name.git
Install dependencies: sudo apt-get install mcrypt
Usage
Bash
Encrypt: bash mcrypt sample.txt

Decrypt: bash mcrypt -d sample.txt.nc

FAQ
What is symmetric encryption?
Symmetric encryption is a type of encryption where the same key is used for both encryption and decryption.
What is mcrypt?
mcrypt is a encryption library for PHP that provides a wide range of encryption algorithms.
